# Git Sync Skill
1. Run `git branch --show-current` and confirm current branch
2. Run `git status` to check for uncommitted changes
3. If uncommitted changes exist, stash them
4. Pull latest from origin for current branch
5. If on a feature branch, merge from main
6. Push current branch to origin
7. If stashed, pop the stash and resolve any conflicts
8. Report final status of all branches
